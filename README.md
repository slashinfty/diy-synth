# diy-synth

## Modules
* PSU
  * Based on [http://musicfromouterspace.com/analogsynth_new/WALLWARTSUPPLY/WALLWARTSUPPLY.php](http://musicfromouterspace.com/analogsynth_new/WALLWARTSUPPLY/WALLWARTSUPPLY.php)
* VCO #1
  * Based on [https://www.ericasynths.lv/shop/diy-kits-1/edu-diy-vco/](https://www.ericasynths.lv/shop/diy-kits-1/edu-diy-vco/)
* VCO #2
  * Based on [https://www.davidhaillant.com/simple-vco-update-1-0/](https://www.davidhaillant.com/simple-vco-update-1-0/)
* LFO
  * Based on [https://www.davidhaillant.com/simple-lfo-1-5/](https://www.davidhaillant.com/simple-lfo-1-5/)
* VCA
  * Based on [https://www.ericasynths.lv/shop/diy-kits-1/edu-diy-vca/](https://www.ericasynths.lv/shop/diy-kits-1/edu-diy-vca/)
* VCF
  * Based on [https://www.davidhaillant.com/ms-20-vcf-clone/](https://www.davidhaillant.com/ms-20-vcf-clone/)
* ADSR
  * Based on [https://benjiaomodular.com/post/2022-02-02-mini-adsr/](https://benjiaomodular.com/post/2022-02-02-mini-adsr/)
* MIDI-to-CV
  * Based on [https://emalliab.wordpress.com/2019/03/02/attiny85-midi-to-cv/](https://emalliab.wordpress.com/2019/03/02/attiny85-midi-to-cv/)
* Sample & Hold
  * Based on [https://www.ericasynths.lv/shop/diy-kits-1/edu-diy-noisesh/](https://www.ericasynths.lv/shop/diy-kits-1/edu-diy-noisesh/)
  * Removed noise and skew circuitry
* Noise
  * Based on [https://web.archive.org/web/20220124122220/http://familjenronnberg.se/~niklas/diy/eurorack/noise/](https://web.archive.org/web/20220124122220/http://familjenronnberg.se/~niklas/diy/eurorack/noise/)
  * Substituted LM324 for CA3140
  * Added random gates with CD4024 based on [http://musicfromouterspace.com/analogsynth_new/NOISECORNREV01/NOISECORNREV01.php](http://musicfromouterspace.com/analogsynth_new/NOISECORNREV01/NOISECORNREV01.php)
* VC Clock & Divider
  * Based on [http://www.modular.fonik.de/Page49.html](http://www.modular.fonik.de/Page49.html)
  * Added gate-to-trigger
* Delay
  * Based on [https://benjiaomodular.com/post/2022-09-11-mini-delay/](https://benjiaomodular.com/post/2022-09-11-mini-delay/)
* Reverb
  * Based on [https://sdiy.info/wiki/CGS_reverb](https://sdiy.info/wiki/CGS_reverb)
  * One channel output
* Logic
  * Original design
  * Two gate inputs with outputs for AND, OR, NAND, NOR, and XOR
* Wavefolder
  * Based on [https://www.ericasynths.lv/shop/diy-kits-1/edu-diy-wavefolder/](https://www.ericasynths.lv/shop/diy-kits-1/edu-diy-wavefolder/)
* Sequential Switch
  * Based on [http://www.modular.fonik.de/Page32.html](http://www.modular.fonik.de/Page32.html)
